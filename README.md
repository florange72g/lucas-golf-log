# GOLF Log

A mobile-first golf performance log for junior competitive golfers. Track every shot, mental game, coach feedback, and build a recruiting report — all from your iPhone.

## Features

- **Dashboard** — Overview, quick actions, recent rounds
- **New Round** — Course setup, tee box, weather, tournament flag
- **18 Hole Entry** — Hole-by-hole tracking (Driver, Fairway, Iron/Wood, Putting, Score)
- **Round Summary** — Scorecard, stats, front/back nine splits
- **Mental Performance** — Rate focus, confidence, composure, and more
- **Coach Reflection** — Strengths, improvements, practice focus, goals
- **Statistics** — Score trends, fairway/GIR %, mental game trends
- **Recruiting Report** — Printable player profile and tournament results

## Getting Started

```bash
npm install
npm run dev
```

Open the URL shown in your terminal (typically `http://localhost:5173`) on your phone or in a mobile-sized browser window.

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- React 19 + TypeScript
- Tailwind CSS 4
- React Router 7
- Local storage persistence

## Design

Fairway green and gold color palette, bottom tab navigation, touch-friendly controls, and iPhone safe-area support.
